# Water Sort Puzzle Game

토스 챌린지 게임 - 워터 소트 퍼즐

[게임 바로가기](https://toss.im/apps-in-toss/game-challenge/neptune)

## 👥 팀원

- **엔젤**: UI/UX 및 애니메이션 담당
- **재은**: 프론트엔드 개발 및 게임 로직 구현

## 🎮 게임 소개

워터 소트 퍼즐은 여러 색상의 액체가 섞인 병들을 조작하여 각 색상별로 분리하는 퍼즐 게임입니다. 플레이어는 전략적으로 액체를 이동시켜 모든 병을 단일 색상으로 채우는 목표를 가집니다.

### 🎯 게임 규칙

- 한 번에 하나의 병에서 다른 병로 액체를 이동할 수 있습니다
- 액체는 위에 있는 것부터만 꺼낼 수 있습니다
- 목적지 병에는 같은 색상의 액체만 쌓을 수 있습니다
- 병의 용량을 초과할 수 없습니다
- 빈 병은 일시적으로 액체를 보관하는 용도로 사용할 수 있습니다

## 🚀 주요 기능

### 게임 로직

- **액체 이동 시스템**: 실제 액체의 물리적 특성을 모방한 이동 로직
- **자동 이동**: 최적의 액체 양을 자동으로 계산하여 이동
- **가능한 이동 탐색**: 현재 상태에서 가능한 모든 이동 경로 탐색
- **게임 완료 검증**: 모든 병이 단일 색상으로 채워졌는지 확인
- **이동 평가 시스템**: 각 이동의 전략적 가치를 평가하는 스코어링

### UI/UX

- 반응형 디자인
- 부드러운 애니메이션 효과
- 직관적인 드래그 앤 드롭 인터페이스
- 진행 상황 시각화

## 🎮 게임 플레이

1. 메인 페이지에서 난이도를 선택합니다
2. 섞인 액체들이 있는 병들을 확인합니다
3. 전략적으로 액체를 이동시켜 색상을 분리합니다
4. 모든 병이 단일 색상으로 채워지면 게임 클리어!
